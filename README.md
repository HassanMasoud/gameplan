# Gameplan

A simple project management application

## Quick Start

Install server dependencies
cd gameplan
npm install

Run React
cd gameplan
npm start

## Built With

* React - Frontend Framework
* Redux - State Management
* NodeJS - JavaScript Runtime
* Firebase - Cloud Database
